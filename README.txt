En esta página web se ha utilizado responsive design gracias a los media queries de CSS,
por lo que gracias a esta tecnologia nuestra página web se adapta al ancho del dispositivo.
También he añadido la cabecera de HTML:
	<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">

Con esto conseguimos que en dispositivos móviles se iterpreten bien los media queries (name="viewport" content="width=device-width)
y con "initial-scale=1, maximum-scale=1" hacemos que los usuarios no puedan hacer zoom a nuestra web
ya que al ser responsive design no es necesario el zoom.

Esto se ha podido comprobar en un dispositivo movil, ya que hemos instalado un servidor web apache en nuestro PC y
hemos conectado el dispositivo móvil a la misma red, así hemos podido acceder a nuestra web.
